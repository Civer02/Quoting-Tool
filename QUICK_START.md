### Step 1: Open the Tool
Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

### Step 2: Configure Company (One-Time)
1. Click **"‚öôÔ∏è Company Settings"**
2. Enter your company name (required)
3. Add contact info (optional but recommended)
4. Upload logo (optional)
5. Set default rates:
   - Labor rate: e.g., $75/hr
   - Equipment rate: e.g., $250/day
   - Tax rate: e.g., 8.5% (if applicable)
6. Click **"Save Settings"**

### Step 3: Create Your First Quote
1. **Customer Info:**
   - Enter customer name (required)
   - Add job name and address (optional)

2. **Scope:**
   - Select a template OR type custom description

3. **Add Labor:**
   - Click "+ Add Labor Item"
   - Select template (optional)
   - Enter hours (use quick-select: 1, 2, 4, 8, 16, 40)
   - Rate auto-fills (edit if needed)

4. **Add Equipment:**
   - Click "+ Add Equipment Item"
   - Select template (optional)
   - Enter days (use quick-select: 0.5, 1, 2, 5, 10, 20)
   - Rate auto-fills (edit if needed)

5. **Notes (Optional):**
   - Select template OR type custom

6. **Generate PDF:**
   - Click "Generate PDF Quote"
   - PDF downloads automatically!
